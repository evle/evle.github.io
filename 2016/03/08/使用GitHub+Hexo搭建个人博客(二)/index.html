使用GitHub+Hexo搭建个人博客(二)

介绍

Hexo is a fast, simple and powerful blog framework. You write posts in Markdown (or other languages)
and Hexo generates static files with a beautiful theme in seconds.

安装

在安装Hexo之前需要先安装以下软件

  • Node.js
  • Git (注意需要添加Git的bin目录至环境变量)
    最后执行以下命令安装Hexo

hexo init
cd
npm install

hexo new <br><figure class="highlight plain"><table><tr><td class="gutter"><pre><span class="line">1</span><br></pre></td><td class="code"><pre><span class="line">3. 写好文章后需要生成网页</span><br></pre></td></tr></table></figure>

hexo generate

hexo server

npm install hexo-deployer-git –save

deploy:
type: git
repo:

hexo deploy –generate

rm -rf .deploy_git/
hexo clean
hexo d -g
```

更换主题

  1. 从https://hexo.io/themes/ 挑选自己心仪的主题并下载

  2. 将下载的主题放至Hexo工程目录中的themes目录

  3. 在_config.yml中修改theme的名字为新主题的名字

If you want to read more details about Hexo, https://hexo.io/docs/
