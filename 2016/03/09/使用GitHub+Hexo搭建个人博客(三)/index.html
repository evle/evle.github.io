草稿功能

在编写长篇文章或者不能及时完成的文章时，可以使用 hexo new draft filename 命令来将文章新建在 _drafts 目录中，_drafts 目录中的草稿文件不会在部署时被上传到Blog. 当文章编写完时, 可以使用 hexo publish draft filename 命令将 _drafts 目录中的文件转移到 _posts 目录中并且进行部署。

添加404页面

在 source 目录下创建一个 404.html 页面，然后通过hexo g -d直接生成页面。404页面只对于绑定了顶级域名的博客有效。

CNAME 存放位置

对于绑定了顶级域名的博客，CNAME文件应该存放在 source 目录中。
