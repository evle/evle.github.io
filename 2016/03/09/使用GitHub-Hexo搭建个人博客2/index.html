使用GitHub+Hexo搭建个人博客(二)

Publish on 2017-12-15

介绍

Hexo is a fast, simple and powerful blog framework. You write posts in Markdown (or other languages) and Hexo generates static files with a beautiful theme in seconds.

安装

在安装Hexo之前需要先安装以下软件

  • Node.js

  • Git (注意需要添加Git的bin目录至环境变量)
    最后执行以下命令安装Hexo

    npm install -g hexo-cli

使用流程

  1. 首先执行以下命令初始化一个Hexo工程.

     hexo init <folder>
 cd <folder>
 npm install

  2. 在新建的HexoProject文件夹中使用以下命令创建文章, 然后用你喜欢的编辑器打开所创建的文章进行书写

     hexo n <title>

  3. 写好文章后需要生成网页

     hexo g

  4. 本地预览
    启动Hexo服务器

     hexo s     # 启动后登陆`localhost:4000`进行预览

布署

布署是将你本地Hexo工程所生成的页面布署到github的一个过程.

  1. 安装hexo布署模块

     npm install hexo-deployer-git --save

  2. 在_config.yml文件中修改一些设置

     deploy:
     type: git
     repo: <repository url>    # e.g., repo: https://github.com/evle/evle.github.io.git

  3. 生成页面并布署至github

     hexo d -g

Tips: 当修改_config.yml文件并且布署后发现网页没有更新时

rm -rf .deploy_git/
hexo clean
hexo d -g

更换主题

  1. 从https://hexo.io/themes/ 挑选自己心仪的主题并下载

  2. 将下载的主题放至Hexo工程目录中的themes目录

  3. 在_config.yml中修改theme的名字为新主题的名字

If you want to read more details about Hexo, https://hexo.io/docs/
