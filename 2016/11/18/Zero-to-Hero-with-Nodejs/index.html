Part 1: Found

Node.js Development Environment Setup

  1. 安装 Homebrew
    在Mac下, 先下载 Homebrew 然后利用 Homebrew 安装 Nodejs 比较方便, 因为 Homebrew 会自动帮你设置环境变量和处理依赖关系之类的. 下载 Homebrew 可以在终端中输入以下命令 ruby -e "$(curl -fsSLhttps://raw.githubusercontent.com/Homebrew/install/master/install)"

  2. 安装 Nodejs
    在终端中运行 brew instll node, 等安装完成后运行 node -v 和 npm -v 提示版本则 Nodejs 安装成功.

  3. 编辑器配置
    推荐使用 Atom 代码编辑器. 配置方法

node 是用来解析并运行 nodejs 代码的程序, npm 是一个 Package 管理工具, 至此基本开发环境搭建完成.
