WebSocket之socket.io

在使用 WebSocket 接口创建应用之前, 我们需要一个实现了 WebSocket 规范的服务器比如Nodejs, 服务器端的编程需要处理以下任务:

  1. 创建HTTP服务器
  2. 路由
  3. 处理路由

创建服务器



window.WebScoket = window.WebSocket || window.MozWebScoket;
if(!window.WebSocket){
    alert('Your browser do not support WebSocket');
    return ;
}

创建服务器

const ws = new WebSocket(‘ws://127.0.0.1:8080’);

数据发送: 客户端 -> 服务器端 (任何JS对象可以通过JSON转换成文本数据进行发送)

webSocket.send(“message”);

事件监听

监听WebSocket打开事件

webSocket.onopen = () => {
console.log(‘Connection established’);
}

客户端接收来自服务器的数据

webSocket.onmessage = (msg) => {
console.log(‘msg’);
}

#监听WebSocket关闭事件
webSocket.onclose = () => {
console.log(‘Connection closed’);
}

监听错误事件

webSocket.onerror = (msg) => {
console.log(‘Error: “ + msg);
}
