Vue 速成(1) 从jQuery到Vue

jQuery: write less, do more
本教程是Vue速成系列第一篇, 介绍如何使用Vue完成那些我们常用jQuery做的事情.

Install:
CDN:

    $ npm install vue
    

<div id="app">
  <button @click="clickMe"></button>
  <input @change="changeText" v-model="name" type="text" placeholder="Enter your name" />
</div>


new Vue({
    el: '#app',
    data: {
      name:''
    },
    methods: {
        clickMe(){
          alert('Hello');
        },
        changeText() {
            alert('Your name is: ' + this.name);
        }
    }
});
