Ionic Crash Course(1) Using Reddit API

Publish on 2017-12-15

The reason for using ionic is simple, build mobile app with html,css and js. In this course, we’re going to build a dead simple app using Reddit API that inspired by Brad Traversy.

Prepare for happy hacking

1. Install inoic
npm install -g cordova ionic
2. Build a inoic app
ionic start myApp tabs
3. Run
cd myApp
ionic serve
