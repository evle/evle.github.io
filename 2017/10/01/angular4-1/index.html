Learn Angular 4 from Scratch(1)

Prepare for happy hacking

1. Install Angular
$ npm install -g @angular/cli
2. Create a Angular App
$ ng new ng-app
3. Run local server
$ ng serve
4. Test build
$ ng build    # Complie entire app -> /dist/

How it work?

  1. app.component.ts: configure a component.

  2. app.module.ts: declar a component.

  1. index.html: using component by selector.

    ng g component videoList

    ngFor ngIf
